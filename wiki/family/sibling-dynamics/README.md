# Sibling dynamics

Relationships and patterns between siblings over time. Not individual incidents — those are `character-moments/` or `memories/`. These pages are about the *shape* of the relationship during some period.

## How to file

- One page per distinct phase or pattern.
- Filename: `YYYY-short-slug.md` (e.g. `2024-robin-following-sam.md`).
- `subjects:` lists the siblings involved.

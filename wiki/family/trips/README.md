# Trips

Family trips — canonical page per trip. Individual kids may have their own `travel/` pages capturing *their* experience of a trip, which link back here.

## How to file

- One page per trip: `YYYY-destination.md` (e.g. `2024-oregon-coast.md`, `2022-thanksgiving-grandparents.md`).
- `subjects:` lists all involved kids plus `family` if the parents were along.
- For big trips, also encourage per-kid `children/<slug>/travel/<same-slug>.md` pages when each kid had distinct experiences.
